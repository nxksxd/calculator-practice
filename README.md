\# CalculatorPractice



\## 📌 Описание проекта



Программа «Калькулятор», разработанная в рамках технологической (проектно-технологической) практики по направлению 09.03.03 «Прикладная информатика», профиль «Геймдизайн и разработка игр».



Приложение реализовано на языке C# с использованием технологии Windows Forms в среде разработки Visual Studio.



---



\## ⚙ Функциональные возможности



✅ Выполнение базовых арифметических операций:

\- Сложение

\- Вычитание

\- Умножение

\- Деление



✅ Поддержка последовательных вычислений  

✅ Кнопки "=" и "C"  

✅ Обработка ошибок (деление на ноль)  

✅ Сохранение результатов вычислений в файл  

✅ Загрузка сохранённых результатов  

✅ Возможность изменения цветовой схемы интерфейса  

✅ Изменение размера шрифта  



---



\## 💻 Технологии



\- C#

\## 🚀 Как запустить программу



\### Способ 1 (рекомендуется)



1\. Открыть файл `CalculatorPractice.sln` в Visual Studio.

2\. Нажать кнопку \*\*Start (▶)\*\* или клавишу \*\*F5\*\*.

3\. Программа запустится в режиме отладки.



---



\### Способ 2 (через сборку)



1\. Открыть проект в Visual Studio.

2\. Выбрать \*\*Build → Build Solution\*\*.

3\. Перейти в папку:

4.\ Запустить файл `.exe`.

---

\## 📁 Сохранение результатов

Файл с результатами создаётся автоматически при нажатии кнопки «Сохранить».

По умолчанию он сохраняется в папке запуска программы (bin/Debug).
