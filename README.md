# 🧮 Calculator Practice

Windows Forms приложение «Калькулятор», разработанное в рамках технологической практики  
по направлению **09.03.03 Прикладная информатика**  
Профиль: *Геймдизайн и разработка игр*

---

## 📌 О проекте

Программа реализует базовые арифметические операции и дополнительные функции хранения результатов.

Разработано на языке **C#** с использованием технологии **Windows Forms** в среде **Visual Studio**.

---

## ⚙ Возможности

- Сложение
- Вычитание
- Умножение
- Деление
- Поддержка последовательных вычислений
- Обработка деления на ноль
- Кнопки "=" и "C"
- Сохранение результата в файл
- Загрузка сохранённого результата
- Смена цветовой темы
- Изменение размера шрифта

---

## 🛠 Используемые технологии

- C#
- .NET
- Windows Forms
- Visual Studio

---

## 🚀 Запуск проекта

### Через Visual Studio

1. Открыть файл `CalculatorPractice.sln`
2. Нажать **Start (F5)**
3. Приложение запустится
